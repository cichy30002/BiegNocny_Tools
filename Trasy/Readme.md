# Narzędzia do tworzenia tras na Biegu Nocnym
### Sposób użycia
1. Pobierz narzędzie - [Generator](Generator_kart_punktow.ipynb) lub [Importer](mapa_google_do_excel.ipynb) i zapisz na swoim Dysku Google.
2. Kliknij w plik -> Otwórz w -> Połącz więcej aplikacji.
3. Znajdź Colabolatory i zainstaluj.
4. Otwórz generator w Colabolatory i postępuj według instrukcji.
